// 03 — FRAUD TYPES

Know the Enemy

These are the fraud categories you'll encounter. Learn to recognize each pattern quickly.

01 — FIFO
Fund In Fund Out
Deposit ≈ Withdrawal ≈ Turnover with minimal Win/Loss. Player doesn't actually gamble — just moves funds using abnormal betting to meet turnover.
02 — AHT
Abnormally High Turnover
Turnover is 100× or more the deposit amount. Example: 17,000 deposit with 1,256,185 turnover. Report to superior immediately.
03 — AB
Abnormal Betting
Betting both sides of an event (Yes/No, Banker/Player, Over/Under) on the same game at the same time. Or betting ALL numbers in Live Casino (Roulette, Fan Tan) to keep Win/Loss minimal.
04 — MAC
Multiple Account Creation
One person creates many accounts. Look for: same username pattern, same name, same DOB, same IP/fingerprint, same password hash, same bonus claimed.
05 — RAF
RAF Bonus Hunter
Player creates multiple fake accounts and refers them to themselves to collect Refer-A-Friend bonuses repeatedly. Username matches Channel Name — strong indicator.
06 — HBF
Hide Balance Fraud
Player hides their bonus balance using Scatter wins across different games (switching games after each scatter) or buying Featured Buy. Makes system think bonus is 0 → Force Close.
07 — CD
Chip Dumping
Two accounts intentionally transfer funds through live casino games — one always wins, one always loses. Same game, same amount, same time, opposite selections (Banker vs Player).
08 — RA
Reload Bonus Abuser
Claims slot-only reload bonuses then plays in other game types (Live Casino, Sports, Fishing) to bypass the required turnover. Bonus gets force-closed by system after new deposit.
09 — FHF
Fingerprint Hiding Fraud
Login shows "No Match Record" for Fingerprint2 and Device Type is "Computer". Player is hiding their device identity to mask multiple account connections.
10 — CF
Suspicious Cockfighting (SV388)
Player bets small amounts then suddenly jumps to large bets with suspiciously high wins in PH Arena. CPC and THA Arenas are exempt — PH Arena is STRICT MONITORING.
11 — CFE
CFE Pending Bets
Cockfight bets showing as "Pending" in report. For new accounts → Hold and escalate. For established accounts with history → release withdrawal is allowed.
12 — CPC
Card PC Abusers
Players depositing continuously via Card PC method with suspicious betting activity. Check for associated fraud patterns before deciding. No fraud → release.
// 04 — ACCOUNT TAGGING

How to Tag Accounts

Before tagging, confirm the account is doing fraudulent activity. Tags restrict what the player can do on the platform.

01
Open Account Profile → Click Edit next to General
The Edit General popup will appear with a list of available tags (Group field).
02
Double-click the tag from the left list
It will move to the right box. You can add multiple tags. Common tags: CRM Exclude Auto Verified and Rs Allow Only Regular Deposit.
03
Click Save to finalize
The tag is now applied. Always tag before proceeding with any further action.
04
Turn Off Bonus (if applicable)
Go to Account Setting → set Allow Receive Bonus to No → Save. ⚠️ Always also add the "Allow Only Regular Deposit" tag so player can still deposit.
⚠️
Important Reminder
When turning OFF bonus rights, you MUST also apply the "Allow Only Regular Deposit" tag — otherwise the player's deposit function may break.
ℹ️
After Tagging — Log to GSheet
Input the account into "Frauds for Monitoring – On Hold" GSheet with Brand, Date, Username, Risk Staff. Then wait for Superior's action: Release or Adjust.
🚫
Adjust Action
If Superior says Adjust: Add a Risk Remark → Reject the withdrawal request → Input account in Adjustment GSheet with full details (Date, Channel, Username, Currency, Status, Adjustment Status, Amount, Risk Code, Risk Staff, Result).
// 05 — ABNORMAL BETTING GUIDE

Spot Abnormal Patterns

Abnormal betting is the root cause behind many fraud types. Here's how to identify it in each game category.

⚽
Sports — Both Side Betting
Signs: Same event, same game, same settled time. Two bets with almost identical amounts but opposite selections (e.g., Yes AND No on same market). Net Win/Loss is always tiny.

Check: Report Tab → View the bet detail → compare eventName, eventID, marketName, selectionName — only matchType differs.
HIGH RISK
🎰
Live Casino — Cover All Outcomes
Signs: Player places stakes on every possible outcome (e.g., all numbers/roads in Fan Tan or Roulette). This guarantees minimal net loss while maximizing turnover.

Check: Click View Icon on the bet → inspect the "bets" JSON field — you'll see all codes covered with equal stake.
HIGH RISK
🃏
One-Time High Amount Betting
Signs: Player suddenly places a single very large bet (e.g., 51,000) unlike all previous normal betting behavior. Often linked to Chip Dumping.

Check: Look for a counterpart account placing the opposite bet at the exact same time with matching bet amount.
INVESTIGATE

// 06 — HIDE BALANCE FRAUD GUIDE

HBF — Key Indicators

Two methods are used — Scatter and Featured Buy. The detection approach is the same for both.

Method 1: Scatter

🔍
Detection
All bets show 0 Bet, 0 Win/Loss but are Free Bets from different games on the same vendor (PG) after the bonus Force Close time.

Method 2: Featured Buy

💸
What is Featured Buy?
Player buys the bonus feature directly from a slot game at a high cost (e.g., 750–900 bet per game), draining their balance without normal play. Each purchased feature from a different game.
🎯
Detection
Sort Win/Loss ascending — large negative bets (750, 900, etc.) from different game names on the same vendor, all on the same day. These are the Featured Buy purchases.
✅
6 Key Factors to Check for HBF
1. Minimum deposit made after bonus claim  |  2. Player bypassing turnover via Featured Buy  |  3. Different games on same vendor, stopping after each Scatter  |  4. Bonus Force Close timestamp  |  5. Losing and winning times on PG specifically  |  6. All bets occurring AFTER the Force Close time
// 07 — MAC & RAF IDENTIFICATION

Finding Linked Accounts

Multiple Account Creation and RAF abuse require checking connections between accounts.

🕵️
Use Fraud Tools
Search the player's username in Fraud Tools. Look at the results for matching: username pattern, full name, DOB, same hashed password, same IP, same bonus claimed. Even 3–4 matching factors = strong MAC signal.
TOOL: FRAUD TOOLS
🖥️
Check IP & Fingerprint
Go to Report → Login Tab. Hover over IP to see "Same IP Users". Hover on Fingerprint2 to see "Same Fingerprint Users". Cross-reference the accounts that appear.
TOOL: LOGIN REPORT
👥
RAF Channel Search
Go to Member → Search → Channel Type: Refer A Friend → enter the Channel Name. Check if all referred accounts have same username/name/IP pattern — that's a RAF ring. Also check if accounts refer each other in a chain.
TOOL: MEMBER SEARCH
🚩
MAC Remark Format
When confirmed: write in Risk Remark — "MAC – Multiple Account Creation with [list of connected usernames]"
